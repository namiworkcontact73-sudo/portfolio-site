## よくある作業

### ▼ 制作実績を追加する

**ステップ1：画像をフォルダに入れてもらう（ユーザーにお願いする）**

まず、ユーザーに以下をお願いしてください：

> 「画像ファイルを以下のフォルダにコピーしてください：
> `wp-content/uploads/` の中に年フォルダ・月フォルダを作って入れてください。
> 例：2026年6月に追加する場合 → `wp-content/uploads/2026/06/画像ファイル名.jpg`
> コピーできたら画像のファイル名を教えてください。」

画像パスが確定したら次のステップへ進みます。

**ステップ2：`portfolio-data.json` に追記する**

`portfolio-data.json` を開いて、一番下の `}` の後に以下の形式で追加します（前の項目との間にカンマが必要です）：

```json
{
  "title": "作品のタイトル",
  "url": "./詳細ページのフォルダ名/index.html",
  "image": "./wp-content/uploads/2026/06/画像ファイル名.jpg",
  "tag": "tag-works-web"
}
```

**タグの種類（どれか1つを選ぶ）：**

| タグ             | 内容              |
| ---------------- | ----------------- |
| `tag-works-web`  | LP・WEB・漫画全般 |
| `tag-works-sns`  | SNS 投稿イラスト  |
| `tag-works-icon` | アイコン制作      |

タグが判断できない場合はユーザーに確認してください。

**ステップ3：ビルドスクリプトを実行する**

```
python build.py
```

を実行してください。`index.html` の制作実績部分が自動更新されます。

完了したら「更新できました。ブラウザで確認してみてください」とユーザーに伝えてください。

### ▼ 制作実績を削除・非表示にする

1. `portfolio-data.json` を開く
2. 該当する `{ ... }` のブロックを丸ごと削除する（前後のカンマも調整する）
3. `python build.py` を実行する

## コンタクトセクション・SP固定CTAボタンについて

### 設置状況

`<section class="contact-section">` と `<div class="sp-fixed-cta">` は **全ページ** に設置されています（お問い合わせページ本体を除く）。

### お問い合わせページのURL（⚠️ 変更予定）

現在のお問い合わせページは `e3daaddaff30447edea8c6b14def12a9/index.html` という WordPress 由来の URL になっています。
将来的にわかりやすい URL（例: `contact/index.html`）に変更する予定です。

**URL を変更した場合は、全ページの以下のリンクを一括で修正してください：**

- `<section class="contact-section">` 内のボタンリンク
- `<div class="sp-fixed-cta">` 内のリンク

> 対象ページが多いため、Python スクリプトで一括置換するのが確実です。前回は `/tmp/add_contact.py` を参考に作業しました。

### コンタクトセクションの内容を変更する場合

テキストや画像を変更する場合も、全ページ同時に変更する必要があります。ページ数が多いためスクリプトでの一括処理を推奨します。
